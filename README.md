NIFS — Nuke It From Space

One action. Clean state. No hesitation.

What NIFS Does

NIFS (Nuke It From Space) instantly clears your current desktop state and replaces it with a stable, believable environment.

- Closes all visible applications immediately
- Removes clutter and active context
- Replaces the screen with a safe, neutral view
- Requires no interaction, no confirmation, no follow-up

Offline Safe Screen

NIFS includes a built-in offline safe screen that works with zero internet connection.

- Live clock and date update in real time
- Headlines rotate subtly over time
- Weather presentation shifts between realistic profiles
- Agenda adapts to time of day
- Status indicators vary slightly

Stable under scrutiny. Boring on purpose.

NIFS Pro — Nuke It From Space (Pro)

Pro adds three features:

- Live Destination Mode
  Open a real-time weather, news, or custom site when internet is available

- Custom Destinations
  Choose your own safe destination or select from predefined options

- Countdown Mode
  Trigger NIFS automatically after a user-defined period of inactivity

Use local when you need certainty.
Use live when you need freshness.

Warning

There is no undo. NIFS is a decisive tool. Use it intentionally.

When NOT to Use NIFS

Avoid triggering NIFS during:

- Operating system updates
- Driver installations
- Firmware updates
- Active file transfers you care about

Usage

Double-click NIFS.vbs to execute.

Do not run NIFS.bat directly.
